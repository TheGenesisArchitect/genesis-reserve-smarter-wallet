First Session Checklist
1. Open the app and sign in.
2. Connect wallet.
3. Open Vaults and select a strategy.
4. Start and complete a test deposit.
5. Verify position appears in portfolio.
6. Open yield section and confirm data loads.
7. Start and complete a test withdraw.
8. Log out and log back in.

Edge-Case Checks
- Disable network mid-flow, then reconnect and retry.
- Background the app for 2 minutes and resume.
- Trigger an expected error and verify message clarity.

How To Report Feedback
Submit each issue with:
- Device + OS version
- App version/build
- Exact steps to reproduce
- Expected result
- Actual result
- Screenshot or screen recording
- Suggested severity (Critical, High, Medium, Low)

Severity Guide
- Critical: app crash, data integrity issue, or blocked core flow
- High: major feature broken with no practical workaround
- Medium: feature works with workaround or partial degradation
- Low: cosmetic or minor UX issue

Success Criteria For Beta Exit
- Core flow completion >= 90%
- No open Critical issues
- High issues are addressed or have approved mitigations
- Crash-free sessions >= 99%

Data and Privacy Notes
- Use only approved test accounts and test funds.
- Do not share private keys or recovery phrases.
- Do not post screenshots containing sensitive wallet details publicly.
